## Dataset
데이터셋은 [SNU factcheck](https://factcheck.snu.ac.kr/)를 크롤링하여 사실, 거짓으로 라벨링하였음.
거짓 데이터셋의 양이 더 많아서 네이버 뉴스에서 추가로 데이터를 모아 두 라벨링된 데이터의 크기가 같게 함.
